# Kafka + Go Consumer

## Structure

```
.
├── docker-compose.yml
└── consumer/
    ├── Dockerfile
    ├── go.mod
    ├── go.sum       # generated by go mod tidy
    └── main.go
```

## Quick Start

```bash
# 1. Generate go.sum (requires Go installed locally)
cd consumer && go mod tidy && cd ..

# 2. Start everything
docker compose up --build

# 3. Open Kafka UI
open http://localhost:8080
```

## Sending test messages

```bash
# Exec into Kafka container
docker exec -it kafka bash

# Produce a few messages
kafka-console-producer \
  --bootstrap-server localhost:9092 \
  --topic app-logs
> {"level":"info","msg":"hello"}
> {"level":"error","msg":"something failed"}
```

## Environment Variables (consumer)

| Variable         | Default               | Description              |
|------------------|-----------------------|--------------------------|
| KAFKA_BROKERS    | localhost:9092        | Comma-separated brokers  |
| KAFKA_TOPIC      | app-logs              | Topic to consume         |
| KAFKA_GROUP_ID   | log-consumer-group    | Consumer group ID        |

## Ports

| Service    | Port  |
|------------|-------|
| Kafka      | 9092  |
| Zookeeper  | 2181  |
| Kafka UI   | 8080  |
